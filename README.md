### 數據處理腳本
- **seismicwave_sort.py**: 
  - **功能**: 讀取 SAC (Seismic Analysis Code) 格式的地震波形檔案，進行帶通濾波、振幅正規化，並繪製地震波剖面圖 (Record Section)。
  - **主要步驟**:
    1. 尋找指定目錄下的 Z 分量 SAC 檔案。
    2. 依測站分組並選取最長筆資料。
    3. 自動取得發震時間與測站距離。
    4. 依震央距排序並繪圖輸出 `record_section.png`。
